# JTCollector WordPress Theme

Custom WordPress theme developed for the **JTCollector sports card website** with integrated WooCommerce support.

The goal of this project is to build a modern, lightweight and maintainable WordPress theme with a focus on clean architecture, Gutenberg compatibility, and a complete e-commerce flow for sports card collectors.

---

## Project Goals

- Develop a custom WordPress theme from scratch
- Implement modern HTML5 and CSS layout structure
- Integrate Gutenberg content blocks
- Prepare full WooCommerce compatibility
- Build a blog system for collector articles and box breaks
- Implement responsive design for desktop, tablet, and mobile
