# JTCollector WordPress Theme

![Deploy](https://github.com/jaroslav-takac/jtcollector-wordpress-theme/actions/workflows/main.yml/badge.svg)

Custom WordPress theme developed for the **JTCollector sports card website** with planned **WooCommerce integration**.

The goal of this project is to build a **modern, lightweight and maintainable WordPress theme** with a focus on clean architecture, Gutenberg compatibility and a full e-commerce flow for sports card collectors.

---

# Project Goals

- Develop a **custom WordPress theme from scratch**
- Implement a modern **HTML5 + CSS layout architecture**
- Integrate **Gutenberg editor content**
- Prepare full **WooCommerce compatibility**
- Build a **blog system** for collector articles and box breaks
- Implement **responsive design** for desktop, tablet and mobile devices

# Development Workflow

The project follows a structured Git workflow using **feature**, **develop**, and **main** branches.

Workflow:

Issue  
↓  
feature branch  
↓  
Pull Request → develop  
↓  
Integration testing  
↓  
Pull Request → main  
↓  
GitHub Actions CI/CD  
↓  
Automatic FTP deployment to Websupport  
↓  
Updated WordPress theme
